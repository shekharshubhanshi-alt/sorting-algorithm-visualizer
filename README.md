# Sorting Algorithm Visualizer

A Python-based desktop application that visualizes popular sorting algorithms using real-time animations. Built with Tkinter to help users understand sorting
logic, comparisons, swaps, and time complexity interactively.

## Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

## Features
- Interactive GUI
- Step-by-step animations
- Dynamic time and space complexity display
- Shuffle and reset controls

## Tech Stack
- Python
- Tkinter

## Learning Outcomes
- Visual understanding of sorting algorithms and time complexity
- Hands-on experience with Python GUI development
- Improved problem-solving and algorithmic thinking

```bash
python sorting_visualizer. py
